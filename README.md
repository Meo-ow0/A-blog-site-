# Collab Blog Project

Welcome to my aesthetic collab blog! This is a space for writers to sign in, share their thoughts, and collaborate on posts.

## Features
- **Secure Login:** Keeps user accounts safe.
- **Aesthetic Design:** A custom, pink-themed interface for a cozy writing experience.
- **Collaborative Posts:** (Add any other features you have here!)

## Technologies Used
- PHP
- MySQL (Database)
- HTML5 & CSS3

## Setup Instructions
1. Clone this repository to your local machine.
2. Import the database file (`database.sql`) into your MySQL server (like XAMPP).
3. Update `db.php` with your local database credentials.
4. Open `login.php` in your browser to start writing!
